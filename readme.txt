The DesignThemes Directory lite addon is a plugin that lets you easily manage and listings on your website.

== Description ==
Easily create and manage listings on your website with this plugin.Packed with loads of features and easy to use.

== Features ==
Our plugin comes with everything you need to create and manage listings.
✔️ Rapidly create property listings.
✔️ Saved property city, country and amenity ( Taxonomies ).
✔️ Archives with three predefined styles.
✔️ MLS Number generation for property. 
✔️ Tested on the major theme frameworks such as https://themeforest.net/user/designthemes/portfolio
✔️ Extensive template tags for customization
✔️ Tons of Hooks & filters
✔️ Compatible with Elementor Plugin ( packed 5+ elementor widgets )
✔️ Compatible with Visual Composer Plugin ( packed 5+ elements )
✔️ Premium version coming soon.


= Install The DT Directory Lite Addon Manually =
1. From the dashboard of your site, navigate to Plugins --> Add New.
2. Select the Upload option and hit "Choose File."
3. When the popup appears select the dt-directory-lite-addon.zip file from your PC.
4 .Activate the plugin through the 'Plugins' menu in WordPress.
